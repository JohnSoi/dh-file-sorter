# DH|FileSorter

## Описание

Консольная утилита для сортировки файлов по расширениям в папки типов.

---

## Требования

На компьютере должен быть установлен [Python](https://www.python.org/downloads/) не ниже 3.12.
Так же следует убедиться, что он добавлен в переменную PATH:
```bash
    python --version
```
Так же следует убедиться, что у вас установлен [uv](https://astral.sh/blog/uv). Он используется для управления зависимостями.

---

## Запуск

1. Для начала необходимо перейти в папку с программой.
2. Открыть терминал в текущей папке.
3. Выполнить команду для создания виртуального окружения:
```bash 
    uv venv
```
4. Установить зависимости:
```bash
    uv sync
```
5. Запустить скрипт:
```bash
    uv run main.py
```
